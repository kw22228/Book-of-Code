### 템플릿 메소드 패턴

#### 커피와 홍차

- 커피 만드는법

  1. 물을 끓인다.
  2. 끓는 물에 커피를 우린다.
  3. 커피를 컵에 따른다.
  4. 설탕과 우유를 추가한다.

- 홍차 만드는법
  1. 물을 끓인다.
  2. 끓는 물에 찻잎을 우려낸다.
  3. 홍차를 컵에 따른다.
  4. 레몬을 추가한다.

#### 템플릿 메소드 패턴의 정의

- 템플릿 메소드 패턴은 알고리즘의 골격을 정의한다. 템플릿 메소드를 사용하면 알고리즘의 일부 단계를 서브 클래스에서 구현할 수 있으며, 알고리즘의 구조는 그대로 유지하면서 알고리즘의 특정 단계를 서브클래스에서 재정의 할 수도 있다.
